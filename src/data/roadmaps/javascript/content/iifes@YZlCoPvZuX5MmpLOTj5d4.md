# IIFE

Immediately-Invoked Function Expression is a function that is executed immediately after it is created.

## Example

```js
// An Async IIFE
( async() => {
    
    const x = 1;
    const y = 9;

    console.log(`Hello, The Answer is ${x+y}`);

})();
```
